Hello!

I'm a human trying to make sense of the complexity of biology. I like guiding experimental work using computational approaches to help understand single microorganisms and their place within microbial communitites. 

* Yeast biotechnology PhD at [BOKU IMMB](https://boku.ac.at/en/btlw/immb); currently at [CEMESS, University of Vienna](https://cemess.univie.ac.at/) as part of the [CoE Microplanet](https://www.microplanet.at/)
* I previously worked with *Komagataella phaffii*, exploring how it adapts to nutrient-limited growth using **genome-scale modelling** and **RNA-seq**. I also applied some **metabolic engineering** tools to improve its ability to produce recombinant protein.
* I am currently enjoying working on some computational projects in microbial ecology including:
  * Developing tools for **microbial community design** to assemble for testing microbiome function — current focus = **nitrogen cycling** in soil.
  * Establishing tools for the investigation of cooccurrence/association of taxa with each other
* I like mining large **metagenomic datasets** to guide community composition and capture core ecological functions.
